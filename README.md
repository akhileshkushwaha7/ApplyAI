# Agentic AI Job Search Automator (n8n Workflow)

This workflow automates the end-to-end job search process using:
- n8n automation
- Gemini + AI Agents
- Google Sheets & Docs
- LinkedIn scraping
- Resume & cover letter generation

## Features
✅ Reads resume  
✅ Scrapes LinkedIn jobs  
✅ Filters by recency, skills & location  
✅ Generates tailored resume + cover letter  
✅ Stores results in Sheets  
✅ Creates Google Doc for each job  
✅ Sends daily email
